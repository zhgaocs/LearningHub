# 常识

## 历史

### 古代

1. 选拔官吏制度
    + 军功爵制
      + 时期：战国
    + 察举制
      + 确立者：汉武帝
      + 特征：地方长官在辖区内考察及选取人才，推荐予上级或者中央，被试用及考核后，再任命为官职
    + 九品中正制
      + 确立者：曹丕
      + 适用时期：魏晋南北朝
    + 科举制
      + 确立者：隋文帝
      + 废除：慈禧，1905年
      + 阶段
        1. 院试：通过者称“秀才”
        2. 乡试：又称“秋闱”，通过者称“举人”，第一名称“解元”
        3. 会试：又称“春闱”、“礼闱”，通过者称“贡士”，第一名称“会元”
        4. 殿试：又称“御试”、“廷试”，通过者称“进士”，分三等
           1. 第一等，进士及第，共三人，即状元、榜眼、探花
           2. 第二等，赐进士出身，第四名到第一百名左右
           3. 第三等，赐同进士出身，若干名
      + 内容
        + 四书五经
        + 八股文

### 近现代

1. 不平等条约
    + 第一个不平等条约：中英《南京条约》（开始沦为半殖民地半封建社会）
    + 割占中国领土最多的条约：中俄《瑷珲条约》
    + 完全沦为半殖民地半封建社会：《辛丑条约》

2. 运动
    + 五四运动 
      + 导火索：1919年巴黎和会中国外交失败
      + 各阶级
        + 决定作用：工人阶级
        + 先锋作用：青年学生
      + 新民主主义革命的开端
    + 一二九运动
      + 时间：1935年12月9日
      + 地点：北平
      + 内容：在中共领导下，北平大中学生数千人举行了抗日救国示威游行，反对华北自治，反抗日本帝国主义，要求保全中国领土完整
      + 意义：标志着中国人民抗日民主运动新高潮的到来，并未之后的中国青年运动指明了正确的政治方向

3. 起义
    + 中国共产党领导 
      + 南昌起义
        + 时间：1927年8月1日
        + 打响了武装反抗国民党反动派的第一枪，拉开了中国共产党独立领导武装斗争和创建革命军队的序幕
      + 秋收起义
        + 时间：1927年9月9日
        + 地点：湖南东部江西西部
      + 广州起义
        + 时间：1927年12月11日
      + 平江起义
        + 时间：1928年7月22日
        + 人物：彭德怀、滕代远
        + 地点：湖南平江

4. 党的会议
    + 八七会议
      + 时间：1927年8月7日
      + 地点：湖北汉口
      + 内容
        + 批判和纠正了陈独秀右倾机会主义错误，确定了土地革命和武装斗争的总方针
        + 毛泽东提出“**枪杆子里出政权**”的论断
    + 古田会议
      + 时间：1929年
      + 地点：福建古田
      + 内容：回答和解决了“**党指挥枪**”等军队建设的一系列基本问题，重申了党对军队实行绝对领导的原则
    + 遵义会议
      + 时间：1935年1月
      + 地点：贵州遵义
      + 内容：纠正王明“左倾”
      + 意义
        + 确立了毛泽东的领导地位，是党的历史上生死攸关的转折点
        + 中国共产党第一次独立地运用马克思列宁主义基本原理解决自己的路线、方针政策的会议，标志着中国共产党从幼稚走向成熟
    + 瓦窑堡会议
      + 时间：1935年12月
      + 地点：陕西瓦窑堡
      + 内容：确立了抗日民族统一战线
    + 洛川会议
      + 时间：1937年8月
      + 地点：陕西洛川
    + 中共七大
      + 时间：1945年
      + 地点：陕西延安
      + 内容
        + 毛泽东作《论联合政府》报告
        + **将毛泽东思想确立为党的指导思想**
    + 中共七届二中全会
      + 时间：1949年3月
      + 地点：河北西柏坡

> [!TIP] 
> 党对军队绝对领导的根本原则和制度，发端于南昌起义，奠基于三湾改编，定型于古田会议。
