# 经济常识

## 价格与货币

### 物价变动

|    项目    |                                 通货膨胀                                  |                                 通货紧缩                                  |
| :--------: | :-----------------------------------------------------------------------: | :-----------------------------------------------------------------------: |
|  物价水平  |                                   上升                                    |                                   下降                                    |
| 货币购买力 |                                   下降                                    |                                   上升                                    |
|    原因    |                     需求过多、成本上升、货币供应过多                      |                     需求不足、生产过剩、货币供应紧缩                      |
|  经济影响  |           短期内可能刺激经济增长，但过高的通胀会引发经济不稳定            |                通常伴随经济萎缩，导致需求不足、失业率上升                 |
|  债务影响  |                      名义债务不变，实际偿还成本下降                       |                      名义债务不变，实际偿还成本增加                       |
|  应对措施  | 央行通过<u>**提高利率**</u>、减少货币供应量、加强金融监管等措施来控制通胀 | 央行通过<u>**降低利率**</u>、增加货币供应量、实施财政刺激等措施来对抗通缩 |

### 货币供应量

|         层次         |                     释义                     |
| :------------------: | :------------------------------------------: |
|          M0          |               M0 = 流通中现金                |
| M1（狭义货币供应量） |      M1 = M0 + <u>**单位**</u>活期存款       |
| M2（广义货币供应量） | M2 = M1 + 单位定期存款 + 个人存款 + 其他存款 |

3. 货币政策工具
    + <u>**公开市场操作**</u>：通过买卖政府证券调节市场流动性
    + <u>**存款准备金率**</u>：调整商业银行需存放在央行的准备金比例，影响其放贷能力
    + <u>**再贴现率**</u>：调整央行向商业银行提供短期贷款的利率
    + <u>**基准利率**</u>：央行设定的基准贷款和存款利率，用于引导市场利率

> [!TIP]
> 1. 逆回购是一种<u>**公开市场操作**</u>工具，指央行向一级交易商（如商业银行）购买有价证券，并约定在未来特定日期将这些证券回售给交易商。通俗地说，这相当于央行向金融机构提供资金，并以有价证券作为抵押。通过这一操作，央行向市场注入资金，从而增加市场流动性
> 
> 2. 贷款市场报价利率（Loan Prime Rate，LPR）
>     + 商业银行对其最优质客户执行的贷款利率，其他贷款利率可在此基础上加减点生成
>     + 期限品种
>       + 1年期
>       + 5年期及以上

## 财政与税收

### 财政

#### 公共财政

+ 定义：政府为满足社会公共需求而进行的收入和支出活动
+ 职能
    + <u>**资源配置**</u>
    + <u>**收入分配**</u>
    + <u>**调控经济**</u>
    + <u>**监督管理**</u>

#### 财政政策


|      项目      |                     财政政策                     |                          货币政策                          |
| :------------: | :----------------------------------------------: | :--------------------------------------------------------: |
|      定义      | 政府通过调整财政收入和支出，以影响经济活动的政策 | 央行通过控制货币供应量、利率和其他信贷条件，调节经济的政策 |
| 制定与执行主体 |             <u>**政府**</u>财政部门              |                            央行                            |
|    主要工具    |           税收<br>政府支出<br>转移支付           |    公开市场操作<br>存款准备金率<br>再贴现率<br>基准利率    |
|    作用机制    |             改变社会总需求来影响经济             |             通过调整货币供应量和利率来影响经济             |

|   项目   |                             扩张性财政政策                             |                  紧缩性财政政策                  |
| :------: | :--------------------------------------------------------------------: | :----------------------------------------------: |
|   目标   |                       刺激经济增长，应对经济衰退                       |            抑制经济过热，控制通货膨胀            |
| 主要措施 | 增加政府支出（如基建、社会福利等）<br>减税，提高居民和企业的可支配收入 | 减少政府支出（削减公共项目、减少补贴等）<br>增税 |
| 预期效果 |                刺激消费和投资，增加总需求<br>降低失业率                |          抑制过度消费和投资，减少总需求          |
| 潜在风险 |                     通货膨胀<br>财政赤字，政府债务                     |         经济增长放缓或衰退<br>失业率增加         |

### 三次分配

|    名词    |                                                                                        解释                                                                                        |
| :--------: | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------: |
| 第一次分配 |                      由市场按照<u>**效率原则**</u>进行，依据生产要素（如劳动、资本、土地等）的贡献确定收入分配。市场在资源配置中发挥决定性作用，强调效率优先                       |
| 第二次分配 |                         由政府主导，侧重<u>**公平原则**</u>，通过<u>**税收、社会保障、转移支付**</u>等手段进行再分配，旨在弥补初次分配的不足，促进社会公平                         |
| 第三次分配 | 主要由企业、社会组织、家庭和个人等基于<u>**自愿原则**</u>，通过<u>**慈善捐赠、资助、义工**</u>等公益方式对财富进行再分配。其特点是自愿性和社会共济性，强调道德力量和社会责任的作用 |

### 国内生产总值（GDP）

1. 国内生产总值（Gross Domestic Product，GDP），是指一个国家或地区所有常住单位在一定时期内生产活动的全部<u>**最终成果**</u>，等于所有常住单位创造的增加值之和。GDP体现的是增加值的生产创造

2. [ ] 名义GDP与实际GDP

### 国民总收入（GNI）

1. 国民收入
    + 工资
    + 利润
    + 利息
    + 租金

2. 国民总收入（Gross National Income，GNI），是指一个国家或地区所有常住单位在一定时期内收入<u>**初次分配**</u>的最终结果，等于所有常住单位的<u>**初次分配**</u>收入之和

$$
\begin{align}
\text{GNI} &= \text{GDP} + \text{来自国外的要素收入净额} \\
&= \text{GDP} + (\text{来自国外的要素收入} - \text{付给国外的要素收入})
\end{align}
$$

### 消费者物价指数（CPI）

1. 消费者物价指数（Consumer Price Index，CPI）是衡量一国一定时期居民消费的商品和服务的<u>**价格**</u>~~（不是价值）~~变化情况的指数。它反映的是普通家庭消费支出的价格变动，通常用于衡量通货膨胀或生活成本的变化

2. 中国CPI构成
    + 食品（权重最高）
    + 烟酒及用品
    + 衣着
    + 家庭设备
    + 医疗保健
    + 交通通信
    + 娱乐教育文化
    + 居住 

### 生产者物价指数（PPI）

1. 生产者物价指数（Producer Price Index，PPI）是衡量生产阶段商品和服务的价格变化情况的指数。它反映的是生产领域的价格波动，主要衡量工业企业生产成本的变化

2. 中国PPI构成
    + 采矿业
    + 制造业
    + 电力、热力、燃气和水生产及供应业
    + 农产品生产 

### 恩格尔系数

+ 食品支出占总消费支出的比重
+ 一般而言，收入越低，恩格尔系数越高；收入越高，恩格尔系数越低

### 基尼系数

+ 衡量收入分配公平程度的指标
+ 数值介于0到1之间。越接近0，收入分配越平均；越接近1收入分配越不均

## 金融体系

### 金融机构

+ 中央银行：中国人民银行
+ 银行业金融机构
  + 政策性银行：政府出资设立，不以盈利为主要目的，而是服务于国家特定的经济和社会发展目标，只有3家
  + 商业银行：以盈利为目的，从事各种商业银行业务，包括吸收存款、发放贷款、办理结算等
    + 国有大型商业银行：国家控股的大型银行，只有6家
    + 股份制商业银行：由国家、企业和社会个人共同出资设立的银行，如中信银行
    + 城市商业银行：主要在城市范围内经营的商业银行，如上海银行
    + 民营银行：由民营资本发起设立的银行，如网商银行
    + 农村金融机构
      + 农村商业银行
      + 农村合作银行
      + 农村信用社
      + 村镇银行
      + 农村资金互助社
  + 银行业专营机构：专门从事某一类特定金融业务的银行机构。例如，专门提供住房贷款的银行
+ 非银行金融机构：通常不提供存款服务，但从事其他金融活动，如投资、保险、证券交易等
  + 证券公司
  + 基金公司
  + 信托公司
  + 保险公司
  + ……
+ 外资、侨资、中外合资金融机构

### 国际储备

1. 国际储备，是一国货币当局持有的，用于国际支付、平衡国际收支和维持其货币汇率的国际间可以接受的一切资产。

2. 主要构成
    + 外汇（中国最主要形式）
    + 黄金
    + 特别提款权（SDR）
    + IMF储备头寸

### 贸易顺差与逆差

|   特征   |                      贸易顺差                      |                   贸易逆差                   |
| :------: | :------------------------------------------------: | :------------------------------------------: |
|   定义   |               出口大于进口，形成盈余               |            进口大于出口，形成赤字            |
| 外汇储备 |                        增加                        |                     减少                     |
|   货币   |                        升值                        |                     贬值                     |
|   影响   | 发生贸易摩擦，可能引发他国贸易保护主义，出现贸易战 | 外债增加，影响国家信用<br>国内产业竞争力下降 |
