# 法律常识

## 宪法

> [!TIP]
> 宪法性法律是指有宪法规范存在其中，但形式上又不具备最高法律效力及严格制定的和修改程序的法律文件

### 宪法修改

1. 提案主体：**全国人民代表大会常务委员**会或**五分之一**以上的全国人民代表大会代表联名提出修改建议
2. 通过条件：全国人民代表大会全体代表的**三分之二**以上多数表决通过

### 地方立法体系

- **地方性法规**：由地方（省、自治区、直辖市、设区的市）**人民代表大会及其常务委员会**制定
- **地方政府规章**：由**地方政府**制定

## 行政法

1. 依法行政六项要求
    + 合法行政
    + 合理行政
      + 公平公正对待原则
      + 考虑相关因素原则
      + 比例原则
    + 程序正当
      + 行政公开
      + 公众参与和回避
    + 高效便民
    + 诚实守信
    + 权责统一

2. 行政行为

| 行政行为 | 种类                                                                         |
| :------: | :--------------------------------------------------------------------------- |
| 行政确认 |                                                                              |
| 行政许可 |                                                                              |
| 行政强制 | 限制公民人身自由<br>查封场所、设施或者财物<br>扣押财物<br>冻结存款、汇款<br> |
| 行政处罚 |                                                                              |

3. 复议前置情形
    + 对当场作出的行政处罚决定不服
    + 行政机关侵犯已依法取得的自然资源权属
    + ​行政机关未履行法定职责
    + 政府信息公开被拒
    + 其他法律、行政法规规定的情形
      + 纳税争议
      + 工伤保险争议
      + 价格违法处罚
      + 审计决定争议

4. 行政诉讼被告
    + 复议机关维持原行政行为，被告是**作出原行政行为的行政机关和复议机关**
    + 复议机关改变原行政行为，被告是**复议机关**

> [!TIP]
> 复议机关改变原行政行为是指复议机关改变原行政行为的**处理结果**

5. 赔偿vs.补偿
    + 赔偿：具有**惩罚性**，通常以过错责任为前提，适用于因违法行为（如侵权行为、违法行政或司法行为）造成的损害
    + 补偿：具有**补充性**，基于公平原则，适用于合法行为（如征收、紧急避险）或不可归责于任何一方的情形（如自然灾害）

## 民法

1. 继承从被继承人死亡时开始
2. 自然人不能成为土地所有权主体，仅国家或集体享有所有权。虽然所有权不可转让，但单位和个人可通过法定程序取得土地使用权（如出让、划拨、租赁等），实现土地资源的市场化利用

## 其他法律

### 社会保险法

1. 社会保险基金
    + 基本养老保险基金
    + 基本医疗保险基金
    + 工伤保险基金
    + 失业保险基金
    + 生育保险基金

2. 社保基金<u>**专款专用**</u>，但可以用于投资（银行存款、国债、股票、证券等，禁止投资房地产、贵金属、P2P等高风险或流动性差的资产）

### 民族区域自治法

1. 民族自治地方的自治机关是自治区、自治州、自治县的人民代表大会和人民政府
