# 法律常识

## 宪法

> [!TIP]
> 宪法性法律是指有宪法规范存在其中，但形式上又不具备最高法律效力及严格制定的和修改程序的法律文件

### 代表选举

1. 直接选举中代表候选人当选需同时满足以下两个条件：
    + ​选举有效条件：选区全体选民的过半数（即超过50%）参加投票
    + ​当选条件：候选人获得参加投票的选民过半数（即超过50%）的选票

### 宪法修改

1. 现行宪法为1982年宪法，并历经1988年、1993年、1999年、2004年、2018年五次修订
2. 提案主体：**全国人民代表大会常务委员**会或**五分之一**以上的全国人民代表大会代表联名提出修改建议
3. 通过条件：全国人民代表大会全体代表的**三分之二**以上多数表决通过

## 立法法

1. 普通行政区的最低立法层级为：**设区的市**和自治州的人大及其常委会

2. 自治县的人大是民族自治地方中最低层级的立法主体，但其立法需报**省级人大常委会**批准后才能生效；而普通县级行政区（非自治县）则不具备立法权

> [!TIP]
> 由地方政府制定的规范性文件被称为**地方政府规章**。其中，设区的市、自治州人民政府是能够制定地方政府规章的最低级别政府

## 行政法

1. 依法行政六项要求
    + 合法行政
    + 合理行政
      + 公平公正对待原则
      + 考虑相关因素原则
      + 比例原则
    + 程序正当
      + 行政公开
      + 公众参与和回避
    + 高效便民
    + 诚实守信
    + 权责统一

2. 行政行为

| 行政行为 | 种类                                                                                                                                                                                                     |
| :------: | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 行政确认 |                                                                                                                                                                                                          |
| 行政许可 |                                                                                                                                                                                                          |
| 行政强制 | 限制公民人身自由<br>查封场所、设施或者财物<br>扣押财物<br>冻结存款、汇款<br>                                                                                                                             |
| 行政处罚 | 警告、通报批评<br>罚款、没收违法所得、没收非法财物<br>暂扣许可证件、降低资质等级、吊销许可证件<br>限制开展生产经营活动、责令停产停业、责令关闭、限制从业<br>行政拘留<br>法律、行政法规规定的其他行政处罚 |

3. 复议前置情形
    + 对当场作出的行政处罚决定不服
    + 行政机关侵犯已依法取得的自然资源权属
    + ​行政机关未履行法定职责
    + 政府信息公开被拒
    + 其他法律、行政法规规定的情形
      + 纳税争议
      + 工伤保险争议
      + 价格违法处罚
      + 审计决定争议

4. 行政诉讼被告
    + 复议机关维持原行政行为，被告是**作出原行政行为的行政机关和复议机关**
    + 复议机关改变原行政行为，被告是**复议机关**

> [!TIP]
> 复议机关改变原行政行为是指复议机关改变原行政行为的**处理结果**

5. 赔偿vs.补偿
    + 赔偿：具有**惩罚性**，通常以过错责任为前提，适用于因违法行为（如侵权行为、违法行政或司法行为）造成的损害
    + 补偿：具有**补充性**，基于公平原则，适用于合法行为（如征收、紧急避险）或不可归责于任何一方的情形（如自然灾害）

6. 公安派出所、具有独立执法主体资格的公安机关内设机构及其工作人员侵犯公民、法人或者其他组织合法权益的，**所属公安机关为赔偿义务机关**

### 物权

1. 物权
    + 用益物权
      + 建设用地使用权
      + ……
    + 担保物权
      + 抵押权
      + 质权
      + 留置权 

2. 自然人不能成为土地所有权主体，仅国家或集体享有所有权。虽然所有权不可转让，但单位和个人可通过法定程序取得土地使用权（如出让、划拨、租赁等），实现土地资源的市场化利用

3. 商标申请注册：初步审定并公告**申请在先**的商标；同一天申请的，初步审定并公告**使用在先**的商标，驳回其他人的申请，不予公告
